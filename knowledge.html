Knowledge

In 2022, I took ITD 110 Web Design I, where I learned the fundamentals of HTML and CSS. This included project planning, structuring HTML, using HTML tags such as !DOCTYPE, html, head, body, and formatting its content with headers, main, titles (h1 to h6), paragraphs, and footers. I also learned to manage media such as images, videos, and audio, create external and internal links, and the basics of tables and forms.

In CSS I learned the differences between inline, internal, and external CSS, and applied various CSS properties such as background, color, text styles, media styles, margins, paddings, and positioning (relative and absolute).
